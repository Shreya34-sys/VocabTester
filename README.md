# VocabTester 
**VocabTester** is an interactive web-based vocabulary learning game. Players can take quizzes, earn XP, coins, track streaks, and level up while mastering new words.

## Features

- User authentication: Sign up, Sign in, or continue as guest
- Interactive multiple-choice quizzes
- Timer for each question
- Leveling system with XP
- Coin rewards and streak tracking
- Daily challenges
- Power-ups and achievements
- Responsive design with animations
